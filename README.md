## 📖 Overview

AI Facial Attendance System is a desktop application that automates attendance using modern face recognition technology.

The system combines **InsightFace**, **OpenCV**, **SQLite**, and **Python** to provide accurate and efficient attendance management for educational institutions.

Unlike traditional attendance systems, it supports multiple attendance modes, intelligent face matching, QR code integration, reporting, and complete student management.

# ✨ Features

## 👤 Student Enrollment

* Register new students
* Capture face directly from camera
* Store student details
* Automatic QR code generation

---

## 🎯 Face Recognition

* AI-powered InsightFace recognition
* Real-time face detection
* High accuracy cosine similarity matching
* Automatic attendance marking

---

## 👥 Group Attendance

* Detect multiple faces simultaneously
* Mark attendance automatically
* Live recognition overlay
* Duplicate attendance prevention

---

## 🎓 Course-wise Attendance

* Filter students by course
* Load only course-specific faces
* Faster recognition
* Separate attendance management

## 📱 QR Code Attendance

* Hybrid Face + QR workflow

# 🚀 Installation

```bash
git clone https://github.com/YOUR_USERNAME/ai-facial-attendance-system.git

cd ai-facial-attendance-system
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```
